## Независимый отчет о рынке онлайн-кредитования и PDL во Вьетнаме за период 2015–2026
Этот репозиторий содержит независимое исследование, основанное на открытых источниках, о развитии рынка онлайн-кредитования во Вьетнаме.

## Темы исследования
- Онлайн-платформы кредитования
- Финтех-компании
- Корпоративные структуры владения
- Трансграничные бизнес-связи
- Платежная инфраструктура
- Взыскание задолженности
- Регулирование
- Защита прав потребителей
